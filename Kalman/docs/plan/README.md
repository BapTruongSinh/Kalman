# Plan Folder

Thư mục này dùng để lưu các plan theo yêu cầu của project owner.

Workflow:

1. User đưa yêu cầu.
2. Viết plan vào `Kalman/docs/plan/`.
3. User review và xác nhận OK.
4. Chỉ sau đó mới bắt đầu code.

Quy ước đặt tên gợi ý:

- `YYYY-MM-DD-short-topic.md`
- hoặc `NNN-short-topic.md` nếu muốn gắn với task cụ thể.
