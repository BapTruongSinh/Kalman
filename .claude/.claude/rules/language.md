# Rule: Ngôn Ngữ Markdown

## Quy tắc mặc định

- Các file `.md` trong dự án sử dụng tiếng Việt có dấu.
- Báo cáo, tóm tắt, cập nhật tiến độ và câu trả lời cho người dùng mặc định dùng tiếng Việt có dấu.
- Có thể giữ nguyên thuật ngữ kỹ thuật tiếng Anh như `pipeline`, `backend`, `frontend`, `Kalman`, `ARX`, `dashboard` khi giúp nội dung chính xác hơn.

## Ngoại lệ

- File skill trong `.claude/skills/` được giữ nguyên theo nội dung hiện có.
- File guide sao chép nguyên bản từ hệ thống khác có thể giữ ngôn ngữ gốc khi cần đối chiếu.
- Khi người dùng yêu cầu rõ ngôn ngữ khác, làm theo yêu cầu mới nhất của người dùng.
