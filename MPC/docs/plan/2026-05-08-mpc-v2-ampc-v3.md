# Plan: MPC v2 và AMPC v3

> Created: 2026-05-08
> Status: Accepted as project backlog input

## Summary

- Tạo folder riêng `demo_kalman/MPC`, độc lập với `Kalman/` nhưng dùng `Kalman` live outputs và `ARX/arx_model.json`.
- V2 là SISO MPC cho một bơm nước, chạy simulation + recommendation.
- V3 nâng lên AMPC bằng bias adaptation, sau đó thêm closed-loop pilot qua HTTP POST + Bearer token.
- Không dùng Hybrid MPC hoặc Hierarchical MPC.

## Key Defaults

- Step: 300s.
- Horizon: 12 steps / 60min.
- Command: `pump_seconds` trong `[0, 300]`.
- Target band: 55-65%.
- Model: reuse ARX `Drip`, map `pump_seconds / 300`.
- Disturbances: measured-hold `Temperature/Humidity/Light`.
- Solver: grid shooting.
- V3 adaptation: bias correction.
- Fail-safe: pump off + alert/log.
- Stale limit: 10 phút.
